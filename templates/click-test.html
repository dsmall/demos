Clicking a filename in the Editor

Normally, clicking a filename in the filetree loads it into the editor, while clicking the name in the location bar (above the edit pane) navigates to that web page. However this behaviour can be modified by holding down the Shift, Cmd or Ctrl keys when clicking, as shown in the following table:

Assumes default browser settings

Load in editor
Replace the content of the edit pane with the selected file. Warning if changes will be overwritten
Replace editor
Navigate to the selected file, replacing the editor in that tab or window
Open in new tab
Navigate to the selected file in a new tab but do not bring to the front
Display in new tab
Navigate to the selected file in a new tab and bring to the front
Display in window
Navigate to the selected file in a window, bringing it to the front
Context menu
Display the browser's context menu (same as right-click)
